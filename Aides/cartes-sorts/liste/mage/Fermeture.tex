\begin{spell}
\spellcarac{name}{Fermeture}
\spellcarac{level}{1}
\spellcarac{duree}{Permanente}
\spellcarac{portee}{3 m}
\spellcarac{description}{\textbf{Durrée :} 2d6 tours

Ce sort bloque magiquement une porte, un portail, une fenêtre, ou tout
autre type de passage fermé.

\begin{itemize}
\item
  \textbf{Ouverture par magie :} Le sort ouverture permet d'ouvrir
  instantanément le portail fermé.
\item
  \textbf{Ouverture par la force :} Les créatures avec au moins 3 Dés de
  vie de plus que le lanceur de sorts peuvent ouvrir le portail bloqué
  au prix d'un round d'effort.
\end{itemize}
}
\end{spell}
